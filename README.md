### Descrição

Projeto focado no desenvolvimento de um produto analítico de séries temporais para previsão do consumo de energia elétrica da classe industrial brasileira. O trabalho utiliza dados reais do IPEADATA (originados pela EPE/Eletrobras) e aplica modelos estatísticos (ARIMA/SARIMA) para capturar tendência, sazonalidade e ciclos presentes na série, contribuindo para o planejamento energético e a estabilidade da malha elétrica nacional.
